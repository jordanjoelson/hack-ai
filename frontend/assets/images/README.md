# Assets

Place your image assets in this folder.

## Required Images

Based on the design, you'll need:

1. **blimp.png** - Orange blimp illustration with stars
   - Used in: Hero section, Cereal box, Final CTA
   - Recommended size: 220x220px (will be scaled as needed)

2. **cloud-star.png** - Cloud with star inside (orange)
   - Used in: Testimonial section as watermark
   - Recommended size: 150px width

3. **cloud.png** - Simple cloud outline (orange)
   - Used in: Testimonial section as watermark
   - Recommended size: 120px width

## Optional Assets

- **schwep-logo.png** - "SCHWEP!" text logo (if you want to use it separately)

## File Naming

Make sure your files match these exact names:
- `blimp.png`
- `cloud-star.png`
- `cloud.png`
